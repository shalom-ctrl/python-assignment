CGPA Calculator (Python)

A simple command-line CGPA calculator built in Python.  
This project helped me practice working with user input, loops, and basic arithmetic operations.

## Features
- Enter multiple courses
- Enter units and grades
- Automatic CGPA calculation
- Input validation

## Concepts Used
- Loops
- Functions
- Conditionals
- Basic arithmetic

## How to Run
python cgpa.py

## Features

- Decrypt text

## Concepts Used
- Strings
- Loops
- ASCII values
- Functions
